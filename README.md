# MedControl API - Gestão de Stock de Medicamentos

## 📌 Problema
A gestão de stock de medicamentos em pequenas clínicas ou farmácias domésticas é muitas vezes feita de forma desorganizada. É difícil saber exatamente o que está disponível, o que precisa de ser reposto ou manter um histórico limpo do inventário. A falta de um sistema centralizado pode levar à compra de medicamentos duplicados ou à falta de itens essenciais em momentos críticos.

## 💡 Solução
O **MedControl API** foi desenvolvido para oferecer uma forma simples, rápida e programática de gerir um inventário de medicamentos. Através de uma interface moderna (FastAPI), o utilizador pode registar novos produtos, listar todo o stock atual e remover itens que já não são necessários. O sistema garante que os dados são validados e armazenados de forma persistente, proporcionando clareza e controlo total sobre o stock de saúde.

## 👥 Público-alvo
Pequenas clínicas, consultórios médicos, cuidadores ou qualquer pessoa que necessite de organizar um stock de medicamentos de forma digital.

## ⚙️ Funcionalidades
- **Listar Medicamentos:** Visualização de todos os itens em stock.
- **Adicionar Medicamento:** Registo de novos itens com validação de dados.
- **Remover Medicamento:** Eliminação de itens do inventário por ID único.
- **Documentação Automática:** Acesso imediato ao Swagger UI para testes rápidos.

## 🛠 Tecnologias
- **Python:** Linguagem base do projeto.
- **FastAPI:** Framework para a criação da API de alto desempenho.
- **Pytest:** Garantia de qualidade através de testes automatizados.
- **GitHub Actions:** Pipeline de CI para validação contínua.
- **Ruff:** Linter para manter a qualidade e padronização do código.
